Projects Overview

This collection represents my work across full-stack systems, cloud-native tooling, and desktop architectures. Each entry reflects a focus on scalability, modularity, and clean design from front to back.

Collections

Tech Stack: React · Django · Node.js · MySQL · OAuth · Amplify · Lambda · API Gateway · RDS · S3

A full-stack project designed to integrate and showcase a diverse range of modern web technologies, cloud services, and authentication systems. Collections served as a sandbox for experimenting with scalable architecture and secure data exchange.

  • Built frontend UI components with React
  • Implemented backend services using Django and Node.js
  • Connected to MySQL databases and integrated OAuth for secure access
  • Used Amazon API Gateway to route requests to Lambda and backend endpoints
  • Deployed via AWS Amplify with Lambda functions and persisted data using RDS
  • Stored static assets and media using S3 for efficient delivery

Rev POS Webapp

Tech Stack: Svelte · FastAPI · Lambda · PostgreSQL · Amplify

A cloud-native point-of-sale (POS) web interface designed to streamline transactions and reporting for small businesses. Built using modular Svelte components and a FastAPI backend, the application leverages AWS services for scalable deployment and persistent storage.

  • Integrated RESTful APIs using FastAPI
  • Deployed frontend/backend via Amplify with Lambda hooks
  • Used PostgreSQL for high-volume transactional data

Rev POS Desktop Application

Tech Stack: Java · FXML · PostgreSQL

A desktop-based POS solution developed in JavaFX, with a declarative UI defined through FXML and visualized via SceneBuilder. Designed for local operation with PostgreSQL, this application supports offline-first workflows with polished interface interactions.

  • Created responsive UI layouts using FXML
  • Mapped user events to Java controller logic
  • Structured PostgreSQL queries for reliable offline data handling
